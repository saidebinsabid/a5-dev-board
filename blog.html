Question-1: What are the different ways to select an element in the DOM?

There are five different way to select an element in the DOM such as, when we know the element has a unique id to use getElementById('') because it's fast and simple, when we want to select multiple elements with the same class to use getElementsByClassName(''), when we want to select all elements of a specific type (all div elements) to use getElementsByTagName(''), when we want to select a single element using a CSS selector (#id, .class, div > p) to use querySelector(''), and when we want to select multiple elements using a CSS selector to use querySelectorAll(' ').

Question-2: What is the difference between innerHTML, innerText, and textContent ?

When we want to get or set the HTML content inside an element to use element.innerHTML = ' ', When we want to get or set the visible text inside an element to use element.innerText = ' ', When we want to get or set the entire text inside an element, including hidden text to use element.textContent = ' '

Question-3: What is event delegation in the DOM?

Event delegation is a technique in Javascript where we add an event listener to a parent element instead of adding it to multiple child elements. Example : document.getElementById('parent'). addEventListener('click', function(event) { if (event.target.tagName === 'BUTTON') { console.log('Button clicked:', event.target.textContent);}})

Question-4: What is event bubbling in the DOM?

Event bubbling means that when an event happens on a child element, it bubbles up to its parent and then to higher ancestors. For example, clicking a button inside a div will trigger the button's event first, then the div's event, and so on. Example : document.getElementById('child'). addEventListener('click', () => { console.log('Child clicked')}) , document.getElementById('parent'). addEventListener('click', () => { console.log('Parent clicked') })

Question-5: How do you create, add, and remove elements using JavaScript?

When we want to create an element to use const newElement = document.createElement('div'); newElement.textContent = 'Hello World';, When we want to Add an element to use document.body.appendChild(newElement);, When we want to Add it before another element to use const reference = document.getElementById('myElement'); document.body.insertBefore(newElement, reference);, When we want to Remove an element to use const element = document.getElementById('myElement'); element.remove();
