Question-1: What are the different ways to select an element in the DOM?

There are five different way to select an element in the DOM such as, when you know the element has a unique id try to user getElementById('') because it's fast and simple, when you want to select multiple elements with the same class getElementsByClassName(''), when you want to select all elements of a specific type (all div elements)getElementsByTagName(''), when you want to select a single element using a CSS selector (#id, .class, div > p)querySelector(''), and when you want to select multiple elements using a CSS selectorquerySelectorAll(' ').
